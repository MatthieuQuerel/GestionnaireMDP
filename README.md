## Installation

Pour installer ce projet, veuillez cloner le dépôt et installer les dépendances nécessaires :

```bash
git clone https://github.com/MatthieuQuerel/myapp.git
cd myapp
pub get
```


## Lancer le Projet

Pour lancer ce projet, exécutez :

```bash
flutter run
```
